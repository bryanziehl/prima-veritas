## [1.0.0] — 2025-11-26
### Added
- Initial public release of **Prima Veritas OSS**.
- Deterministic normalization pipeline (Node 18.x).
- Deterministic K-Means implementation with canonical init.
- FullProof runner for dataset-level reproducibility.
- Docker environment for cross-machine determinism.
- Canonical goldens for the IRIS and Wine datasets.
- Repository structure and dataset organization.
- FITGEN digest builder (tools/).
- Complete MIT License.
